### 三、mysql的简单操作

建立数据库：mysql> CREATE DATABASE 库名

删除数据库：mysql> DROP DATABASE 库名

建立数据表：mysql> USE 库名;mysql> CREATE TABLE 表名 (字段名 VARCHAR(20), 字段名 CHAR(1))

删除数据表：mysql> DROP TABLE 表名
